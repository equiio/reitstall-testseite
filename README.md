# Reitstall-Testseite

Eine einfache statische Test-Webseite aus HTML und CSS. Sie benötigt weder Datenbank noch Server-Software und kann direkt mit GitHub Pages veröffentlicht werden.

## Dateien

- `index.html` enthält Texte und Aufbau.
- `style.css` enthält Gestaltung und Smartphone-Anpassungen.

## Veröffentlichung

Unter **Settings → Pages** als Quelle **Deploy from a branch** wählen. Anschließend den Branch **main** und den Ordner **/(root)** auswählen und speichern.
